# TableViewDelegation-Swift
App demo for basic UITableView delegation 

- Swift 2.2
- Xcode 7.3

##Features:
+ TableView w/prototype cell in interface builder 
+ utilization of UITableViewDelegate and UITableViewSource protocols in the main ViewController
+ ability to add and delete cells
